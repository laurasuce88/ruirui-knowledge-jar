Chapter 01: 模型本质
01 / ESSENCE

LLM 本质

概率猜词游戏：核心是通过训练海量文本，学习预测下一个 Token 的概率分布，从而模拟人类语言表达。

02 / CORE

自注意力机制

通过权重分配实现“划重点”，约束回复逻辑，确保模型在处理长文本时不会偏离主题，实现上下文理解。

03 / POWER

算力与推理

参数规模突破临界点带来的“涌现”能力，让模型不仅能生成文本，更能理解复杂的语法、代码逻辑与抽象推理。

Chapter 02: 外置大脑 (RAG & 向量检索)
01 / EXTENSION

RAG (时效补丁)

解决模型训练数据的滞后性，通过检索外部知识库或联网搜索，提供最新且符合预期的回复。

02 / TRANSLATION

向量化 (数字翻译)

将模型看不懂的文字转化为坐标向量，通过计算语义相关性进行精准匹配输出。

03 / MEMORY

上下文窗口 (瞬时记忆)

模型一次能处理的信息上限。RAG 通过精简信息，补充了模型记忆长度的局限。

Chapter 03: 指挥官与行动派 (Prompt & Agents)
01 / PROMPT

PE 工程 (精准导航)

核心是让模型在约束下产出预期回答。虽然目前需要明确角色、目标和 Few-shot，但随着模型理解力提升，未来将趋向于更自然的语音交互。

02 / LOGIC

CoT 思维链 (思考模式)

强制模型展开推理过程，一步步执行以减少误差。o1/r1 等模型已自带“思考模式”，大幅提升了逻辑复杂任务的成功率。

03 / AGENTS

AI Agents (数字员工)

Agent = 模型 + 规划 + 工具 + Skill + 记忆 + 调度。如 OpenManus 等智能体正通过 IM 入口深度融入办公，成为真正的数字员工。

Chapter 04: 破次元壁 (多模态与世界模型)
01 / EXTENSION

多模态能力 (能力外延)

突破文字局限，实现对图片、视频、语音的深度理解。Gemini 与豆包系列模型正通过多模态交互大幅提升模型能力的外延。

02 / PHYSICS

世界模型 (物理逻辑)

让模型理解重力、碰撞、遮挡等物理规律。Sora 与 SeedDance 2.0 等视频模型标志着 AI 正在向理解物理真实世界迈进。

03 / SCALING

Scaling Law (规模涌现)

当计算量和资源积累到临界点，模型能力会出现“涌现”现象。这种提升不是线性的，而是非线性的、跨越式的进步。

Chapter 05: 赋予灵魂 (微调与对齐)
01 / SFT

SFT (监督微调)

通过提供标准的“问题-回答”对（SFT 数据集），让模型学会特定的对话格式和任务技能，开启“格式化模仿”模式。

02 / RLHF

RLHF (强化学习)

人类通过给不同模型的回答进行打分排序，建立奖励机制，引导模型给出更符合人类价值观和预期的“好答案”。

03 / DPO

DPO (偏好优化)

采用更直接的“二选一”对比方式，跳过复杂的奖励模型环节，实现模型能力与人类偏好的快速迭代与对齐。

Chapter 06: 更巧更省 (模型效率与架构)
01 / MOE

MoE (混合专家)

像 DeepSeek 一样“按需分配专家”，只激活必要的模块，实现模型规模与算力成本的最优博弈。

02 / QUANTIZATION

量化技术 (精度换空间)

压缩模型权重，让大模型能在手机、Mac 等端侧设备上流畅运行，实现 AI 的私有化部署。

03 / KV CACHE

KV Cache (键值缓存)

存储中间计算结果，避免推理过程中的重复劳动，让 AI 回答实现“秒级响应”。

Chapter 07: 跨越屏幕 (具身智能)
01 / ROBOTICS

具身智能 (机器人赛道)

给大模型装上“身体”，让 AI 不仅能聊天，更能在物理世界中感知环境并精准完成复杂任务。

02 / ACTION

VLA 模型 (行动进阶)

视觉-语言-动作模型。作为多模态能力的进阶，它实现了从“看图说话”到“看图发指令”的质变，让机器人能理解并执行具体动作。

03 / SIMULATION

Sim-to-Real (虚拟进阶)

仿真到现实。在虚拟世界中高效率训练机器人，再将训练成熟的“最强大脑”无缝迁移到物理实体中，大幅降低研发成本。

Chapter 10: 感知层设计 (Perception) 🍓🍑🥑
01 / INSIGHT 🐼

🍓 核心洞察

感知层 (Perception) 是 Agent 的“中枢神经系统”，通过主动捕捉环境变化并进行实时建模，为决策提供高质量的上下文背景。🐨

02 / LOGIC 🐱

🍑 交互逻辑

1. 🐣 主动捕捉：深度关联记忆；2. 🍼 多模态融合：全方位理解；3. 🍭 上下文觉醒：基于业务深度的智能建议。

03 / ACTION 🍡

🥑 未来行动

强化 L2 记忆层的业务洞察力；设计多维实时风险雷达；提供从感知到解释的全栈透明决策能力。🍒

Chapter 11: 大脑与规划 (Brain & Planning) 🧠🗺️🚀
01 / REASONING 🧠

潜空间推理 (JEPA)

联合嵌入预测架构。通过在抽象的潜空间（Latent Space）而非像素空间进行预测，让模型避开无关细节，专注捕捉核心语义与因果逻辑。🌌

02 / MODEL 🗺️

世界模型 (V-JEPA 2)

Meta 视觉世界模型进化。具备更强的自监督预测能力，让 AI 能在观察视频时通过内部模拟学习物理规律，像人类一样理解世界的运转方式。🌍

03 / COMPUTE 🚀

测试时计算 (TTC)

Test-Time Compute。让模型在回复前通过系统 2 (Slow Thinking) 进行反复博弈、搜索与验证，实现“思考越久，结果越准”的智能跨越。💎
